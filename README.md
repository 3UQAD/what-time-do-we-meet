# When do we meet?

*Find a shared moment across cities.*

A small, mindfully vibe-coded (if that's a thing) tool to compare times across cities and meet in one moment. Runs in the browser; share a link and the same cities open for everyone.

Time is one of the things that separates us—borders, distance, and different time zones make it harder to meet. This app is here to help with **coordination** for teams and communities scattered across places. 

## Features
- Data from [city-timezones](https://github.com/kevinroberts/city-timezones) - thanks! :D)
- URL encodes the city list: copy the link to share the same setup.
- Light/dark theme (manual toggle; optional auto from base city day/night).
- 24h/12h, keyboard nudge on clocks, optional time picker.
